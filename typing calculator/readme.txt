# Typing Speed Test Application

## Overview
The **Typing Speed Test Application** is a graphical user interface (GUI) tool  that allows users to test and improve their typing speed. The application calculates words per minute (WPM), tracks typing errors, and records the history of past typing tests.

## Features
- Randomly selects a typing test passage from a predefined list.
- Allows users to enter custom text for personalized practice.
- Displays real-time typing speed (WPM), errors, and time taken.
- Maintains a history of past typing test results.
- Provides a reset functionality to start a new test.
- User-friendly interface with a dark theme for comfortable usage.

## Technologies Used
- **Programming Language:** Python
- **GUI Framework:** Tkinter
- **Other Libraries:** `random`, `time`

## How to Run the Application
1. Ensure you have Python installed on your system.
2. Save the script as `typing_speed_test.py`.
3. Open a terminal or command prompt and navigate to the script's directory.
4. Run the script using the command:
   ```sh
   python typing_speed_test.py
   ```
5. The application window will open, allowing you to start the typing test.

## Usage Instructions
1. Click **"Start Test"** to generate a random passage.
2. Begin typing in the input field as accurately and quickly as possible.
3. The application will calculate and display your WPM, errors, and time taken.
4. Click **"Reset"** to save your test history and start a new test.
5. (Optional) Enter a custom text in the provided input box and click **"Use Custom Text"** for a personalized test.
